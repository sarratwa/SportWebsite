# SportWebsite
A Website to keep track of current scores in matches as well as team comps

## Description
##### User Interface
- Create Account Page
- Login Page
- Home Page
- Teams Page
- Matches Page
- Current Match Page

##### Admin Panel
- Add Match Page 
- Display Matches Page 
- Add Team Page 
- Display Teams Page 
- Current Match update Page 

## Technologies
- Angular
- NodeJS
- ExpressJS
- MongoDB
